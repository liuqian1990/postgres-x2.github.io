15.3. Getting The Source

Note: XCONLY: The following description applies only to Postgres-XC.

The Postgres-XC 1.2.1 sources can be obtained from its Web site or development site. After you have obtained the file, unpack it: 

gunzip pgxc-v1.2.1.tar.gz
tar xf pgxc-v1.2.1.tar

This will create a directory pgxc under the current directory with the Postgres-XC sources. Change into that directory for the rest of the installation procedure.

You can also get the source directly from the version control repository, see Appendix I.
