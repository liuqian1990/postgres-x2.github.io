15.6. Supported Platforms

Note: XCONLY: The following description applies only to Postgres-XC.

Postgres-XC has been tested on Intel-based CPU mainly. Other CPU architectures may also work but are not currently being tested.

Note: XCONLY: The following description applies only to Postgres-XC.

Postgres-XC can be expected to work on these operating systems: Linux (all recent distributions), FreeBSD and Mac OS X. Other Unix-like systems may also work but are not currently being tested.

If you have installation problems on a platform that is known to be supported according to recent build farm results, please report it to <postgres-xc-bugs@lists.sourceforge.net>. If you are interested in porting Postgres-XC to a new platform, <postgres-xc-developers@lists.sourceforge.net> is the appropriate place to discuss that.
